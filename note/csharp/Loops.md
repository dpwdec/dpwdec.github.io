## For

You can **define multiple variables of the same type** in a `for` loop signature by `,` comma separating them.
```csharp
for (int x = 0, y = 0; x < 10; x++)
{
  GetCoord(x, y + x/2);
}
```

You can **define compound boolean expressions** in a `for` loop's conditional field.
```csharp
for (
```
